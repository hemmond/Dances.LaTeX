{\large Progresivní reel pro 4+2n tanečníků ve Walls of Limerick formaci}\hfill

Dva páry stojí proti sobě, ženy napravo od partnerů. 
Tanec je progresivní, takže každý pár postupuje stále stejným směrem, dokud nedorazí na konec longwaye. 

Tento tanec je uveden ve sbírce \Source{Tricitka}

\begin{description}
    \danceFigure{A}{Advance \& Return \bars{4}}{
        \item[1–2]Advance promenádním krokem vpřed, na konci se otočit čelem vzad
        \item[3–4]Advance promenádním krokem domů, na konci se otočit čelem vzad
    }

    \danceFigure{B}{Right \& Left Half-turn \bars{4}}{
        \item[1–2]Páry zátočka za pravou ruku o půlku
        \item[3–4]Páry zátočka za levou ruku o půlku
    }

    \danceFigure{C}{Sidestep \& Heyes \bars{16}}{
        \item[1–4]Sidestepem výměna partnerů v páru, záskoky
        \item[5–6]Dámy výměna míst promenádním krokem, míjet levým ramenem
        \item[7–8]Pánové výměna míst promenádním krokem, míjet pravým ramenem
        \item[9-16]Dtto 1-8
    }

    \danceFigure{D}{Teleskop jedničky \bars{8}}{
        \item[1–4]První pár sidestep dolů středem (drží se za pravou ruku), druhý pár sidestep nahoru vnějškem, záskoky při kterých se první pár pustí a druhý chytí
        \item[5–8]Druhý pár sidestep dolů středem (drží se za pravou ruku), první pár sidestep nahoru vnějškem, záskoky
    }

    \danceFigure{E}{Hvězda za pravou a zpět \bars{8}}{
        \item[1–4]Oba páry hvězda za pravou ruku, promenádním krokem
        \item[5–8]Oba páry hvězda za levou ruku, promenádním krokem, domů
    }

    \danceFigure{F}{Boční teleskop \bars{8}}{  
        \item[1–8]Dtto figura D, akorát místo prvního páru je to první pán s druhou dámou
    }  

    \danceFigure{G}{Hvězda za levou a zpět \bars{8}}{
        \item[1–4]Oba páry hvězda za levou ruku, promenádním krokem
        \item[5–8]Oba páry hvězda za pravou ruku, promenádním krokem, domů
    }

    \danceFigure{H}{Swing Out \bars{8}}{
        \item[1–8]Swing v kříženém držení o půlku -- progrese
    }  
\end{description}
